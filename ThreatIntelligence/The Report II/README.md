## 💡 Lessons Learned

* Clear separation between NOC, SOC, and ISCM is essential for efficiency.

* The SOC should focus only on detection and response, not IT operations or compliance.

* Intelligence-driven monitoring is more effective than relying only on tools.

* SOC structure must be aligned with organization size (e.g., distributed models for larger orgs).

* Incident prioritization helps focus on high-impact threats.

* Adequate data retention is important for forensic investigations.

* Continuous monitoring (ISCM) improves risk awareness.

* Effective security requires a balance of people, process, and technology.

* Automation supports analysts but cannot replace human expertise.

* Cross-team collaboration is critical for strong cybersecurity operations.
